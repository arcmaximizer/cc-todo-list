# Contributing to Todo List App

Welcome! This repo is for learning collaborative Git workflows.

## Workflow

1. **Pick an issue** or create one if there's a bug/feature
2. **Create a branch**: `git checkout -b feature/description` or `fix/description`
3. **Make changes** and commit with clear messages
4. **Push**: `git push -u origin your-branch-name`
5. **Open a PR** and request review

## Branch Naming

- `feature/add-todo` - for new features
- `fix/delete-todo` - for bug fixes

## Commit Messages

Use clear, concise messages:
- "Add todo input component"
- "Fix delete button styling"

## Code Style

- Use TypeScript (.tsx)
- Use plain CSS (no Tailwind)
- Keep components small and focused

## Getting Help

Ask in your coding club or open an issue!
