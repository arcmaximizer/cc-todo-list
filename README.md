# Todo List App

A simple todo list application built with React, TypeScript, Vite, and plain CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 to view the app.

## Project Structure

```
src/
├── components/
│   ├── TodoInput.tsx    # Input form for adding todos
│   ├── TodoItem.tsx     # Individual todo item
│   └── TodoList.tsx     # List container
├── App.tsx              # Main app component
├── App.css              # Global styles
└── main.tsx             # Entry point
```
